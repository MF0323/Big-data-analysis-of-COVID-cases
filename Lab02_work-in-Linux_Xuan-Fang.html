Exploration of variable age:

The age variable must contains some unusual (non-numeric) values in it because it is read in as a character column.

head(cases$age[grep("[a-zA-Z]", cases$age)], 10)

Commentary:

There are text characters contained in the age variable. It shows that the age for some young patients has been recorded with a time unit (weeks/month/months).

An easy approach is to coerce character value to numeric, which will generate NA values for any ages that contain a time unit.

## Coerce to numeric
years <- as.numeric(as.character(cases$age))

Task 5: Split the data 80/20 into training and test sets

Make a random split, with 80% of the data in the training set.

set.seed(123)
train <- sample(seq(N), N*0.8)

yTrain <- logdays[train]
xTrain <- years[train]

The test set contains the remainder of the data.

yTest <- logdays[-train]
xTest <- years[-train]

Commentary:

  • We use the set.seed() function to make sure our procedure is reproducible.

  • Then, we use the sample() function to choose a sample from the numbers of \(339,547\) rows from the initial data set, and the size of our sample is 80% of the number of rows in the initial data set.

  • Finally, we use the selected rows (in the previous step) to subset the original data set as the training set and those non-selected rows to subset the original data set as the test set.

In the training data set, the relationship between logdays and years:

par(mfrow=c(1, 2))

Conclusion

  • The data consist of around 340,000 individual COVIS cases spread across six CSV files. We looked at the relationship between patient age and the probability of death and found some evidence that risk of death increases with age.

  • One reason for caution is that we have much less data for people at advanced ages, so our estimate of the risk of mortality where the risk becomes higher is not very reliable.

  • An additional problem is that the data contain censored observations. There may be patients in the data set who did not survive, but were still alive when the data set was recorded.
