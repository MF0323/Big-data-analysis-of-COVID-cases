10-fold cross-validation

Use 10-fold cross-validation to fit a logistic model to predict the probability of death based on age.

Step 1: Randomly shuffle the data

## Set the seed
set.seed(123)
## Randomly shuffle the data
mydata <- mydata[sample(nrow(mydata)),]

Step 2: Create 10 equally size folds & LogLoss function

## Step 2.1: Create 10 equally size folds
folds <- cut(seq(1, nrow(mydata)), breaks = 10, labels = FALSE)

## Step 2.2: Create the `LogLoss` function 
## Evaluate ('y' should be 1 or 0 [or TRUE/FALSE])
LogLoss <- function(p, y) {
    -mean(y*log(p) + (1 - y)*log(1 - p))
}

Step 3: Perform 10-fold cross-validation

out <- numeric(10)
simple <- numeric(10)
### Using for loop to repeat the procedure
for(i in 1:10){
  # Create training set by using the folds not in the fold i
  train <- mydata[folds!=i, ]
  # Create test set by using the fold i
  test <- mydata[folds==i, ]
  ### Use the test and train data
  ### For set i, using the all other 9 sets to fit the logistic model
  fitGLM <- glm(deceased ~ years, data = train, family="binomial") 
  ### Predict the the probability of death based on age using the test set (fold) i 
  predGLM <- predict(fitGLM, data.frame(x=test["years"]), type="response")
  ### Then, calculate the log-loss value 
  # For the model
  out[i] <- LogLoss(predGLM, test$deceased)
  # For the simple overall proportion
  predProp <- mean(train$deceased, na.rm=TRUE) 
  simple[i] <- LogLoss(predProp, test$deceased)
}

mean(out) # logistic model

The log-loss value for the logistic model is lower than the log-loss for a model that only assumes a constant risk of death for all ages, indicating that the logistic model is capturing some of the patterns in the data (i.e., there is some evidence that the risk of death increases with age).

The variability of the proportion of patients who do not survive is much greater at advanced ages, but there is a clear trend of an accelerating risk with advanced age, which is captured much more effectively than by a horizontal line (the constant-risk model, represented by the grey line).

Conclusion

  • The data consist of \(99,975\) individual COVID-19 cases in a CSV file. We looked at the relationship between patient age and the probability of death and found some evidence that risk of death increases with age.

  • We evaluated the model on the test set (by using 10-fold cross-validation) and compared its performance to a simple overall proportion. The predictions based on our model seem to perform better (smaller log loss) compared to the one using the simple overall proportion. Our model captures most of the data, but some observations around age \(90\) to \(105\) are left behind.

  • One reason for caution is that we have much less data for people at advanced ages and we lose about one third of deceased cases when created years variable for our analysis. In that case,our estimate of the risk of mortality is not very reliable.

An additional problem is that the data contain “censored” observations. There may be patients in the data set who did not survive, but were still alive when the data set was recorded.
